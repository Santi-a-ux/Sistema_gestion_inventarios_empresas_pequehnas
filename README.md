# Sistema de gestion

## Dependencias 
- pip install uvicorn
- pip install tkinter
## Como correr

### frontend 
- bienvenida.py (se corre normalmente)
- index.html (se corre con live server o como quede mas comodo)

### backend 
- main.py (se corre con (uvicorn main:app --reload) o (python -m uvicorn main:app --reload
)
)
- luego se visuliza en este link del servidor ([text](http://127.0.0.1:8000/docs))
